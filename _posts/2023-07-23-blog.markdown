Git 설치는 구글검색하면 잘 나와있습니다.

```
$ git config --global user.name "나의 사용자 이름"
$ git config --global user.email "내 이메일 주소"
```

$ git config --global user.name만 입력하면 지금 설정된 name을 알 수 있습니다.
